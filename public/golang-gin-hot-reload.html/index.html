然后搜索一把发现有不少推荐 realize 这个工具的, 刚开始看文档半天都不知道这是怎么个玩意, 用法总是说的模棱两可的, 捣腾一番也算能用了. 吐槽文档的其实也不少的.

不过其实要做的功夫也很少, 在项目根目录初始化一个 .realize.yaml

settings:
  legacy:
    force: false
    interval: 0s
schema:
- name: whatever
  path: .
  commands:
    run:
      status: true
  watcher:
    extensions:
    - go
    paths:
    - /
    scripts:
      - type: after
        command: ./your_binary_file_name
        output: true
    ignored_paths:
    - .git
    - .realize
    - vendor

就完事了, 讲真那堆参数大部分都是靠蒙的 = . =

启动就跑

realize start --run

结果就是酱紫
